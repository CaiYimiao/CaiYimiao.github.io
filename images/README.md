# Images folder

Drop your image files here using these **exact filenames** and they'll appear automatically
on the site (until then, styled placeholders are shown).

## Profile / hero
- `profile.jpg` — your headshot (square works best, e.g. 600×600)

## Projects (Projects section)
- `project-roboverse.jpg`
- `project-jig.jpg`
- `project-robotstudio.jpg`
- `project-cogs.jpg`

## Certificates (Certifications & Awards section)
- `cert-roboverse.jpg`
- `cert-nyaa.jpg`
- `cert-edusave.jpg`
- `cert-linkedin.jpg`

## Gallery (Gallery section)
- `photo-1.jpg` … `photo-6.jpg`

Tip: JPG or PNG both work — just keep the filename (the part before `.jpg`) the same.
Aim for files under ~500 KB each so the page loads fast.
